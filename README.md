# Hi, I'm Michał! 👋

### Fullstack Developer | Angular & .NET Enthusiast | MSc Cybersecurity Student

My primary area of expertise is building scalable, secure, and modern web applications. My biggest passion lies within the **.NET** ecosystem on the backend and **Angular** on the frontend - in my opinion, it is one of the most powerful combinations for modern enterprise software development.

As a Computer Science student specializing in **Cybersecurity** at the Cracow University of Technology, I always put a strong emphasis on data security, software architecture, and development best practices (Clean Architecture, SOLID).

---

### 🚀 Core Interests & Focus

* **Frontend:** Advanced Angular architecture (RxJS, Signals, State Management), rendering performance optimization, and clean code written in TypeScript.
* **Backend:** Designing high-performance APIs with .NET, asynchronous programming, message brokers (RabbitMQ), and microservices architecture.
* **Security:** Web application security (OWASP Top 10), cryptography, secure authentication/authorization (OAuth2/OIDC), and DevSecOps.
